# cv_utils

## 为什么写这个文件？

写这个工具的初衷是使得图像处理能够变得更简单。

## 特点

1. 完全不需要编写代码，只需要更改配置文件当中的数据，处理步骤，输入输出定义好的话，就能够完成图像的批量处理任务。

2. 可扩展性：只要继承`CVBase`这个类，然后重写需要的方法，在方法前面加上下划线，就能够完成相应的操作。

3. 写了一个替代`print`的方法，重命名为`dump`，能够显示更加多彩的画面。

## 文件用法：

### 主要关键字用法：

input_label:    输入文件的标签
output_label：  输出文件的标签
verbose：       显示处理信息。
